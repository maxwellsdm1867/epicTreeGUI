# EpicTreeGUI

A MATLAB GUI for visualizing and browsing epoch tree data structures.

## Overview

EpicTreeGUI provides an interactive interface for exploring hierarchical epoch data, commonly used in electrophysiology and neuroscience research.

## Features

- Visualize epoch tree structures
- Browse hierarchical data
- Navigate through epochs and their properties

## Requirements

- MATLAB R2019b or later
- No additional toolboxes required (base installation)

## Getting Started

1. Clone this repository
2. Open MATLAB and navigate to the repository directory
3. Run `epicTreeGUI` to launch the application

## Structure

- `epicTreeGUI.m` - Main GUI application
- `src/` - Source code and helper functions
- `examples/` - Example data and usage scripts
